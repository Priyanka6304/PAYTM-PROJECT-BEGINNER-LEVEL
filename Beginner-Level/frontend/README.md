# PAYTM WALLET DEVOPS PROJECT

A complete full-stack Paytm Wallet application deployed using Docker, Kubernetes, Minikube, and AWS EC2.

---

# PROJECT OVERVIEW

This project demonstrates:

- Frontend deployment using React
- Backend deployment using Node.js + Express
- MySQL database integration
- Docker containerization
- Kubernetes deployments and services
- Minikube cluster setup
- AWS EC2 deployment
- Docker Hub image management
